[トイレ時刻の記録方法]
　[大], [小], [大小]のボタンをクリックしますとトイレ時刻を記録します。

[トイレ時刻の追加]
　[データ追加]をクリックしますと手動でトイレ時刻を追加できます。

[記録したトイレ時刻の修正方法]
　トイレ時刻一覧の修正したい項目をダブルクリックします。
　時刻修正画面が表示されます。記録したデータを修正後、[変更]クリックして修正されます。

[記録したトイレ時刻の削除方法]
　トイレ時刻一覧の削除したい行を選択して[DEL]キーを押します。削除確認があります。


現在のクリック後の時間とトイレ種別を表示
前回の大からの経過時間を表示

[大], [小], [大小]のボタンをクリック後は１分間は続けてクリックすることはできません。

トイレチェックしたデータは、実行ファイル[ToiletChecker.exe]と同じフォルダにあります。
ファイル名は、[ToiletChecker.txt]です。
CSV形式のファイルです。
2016/04/01 06:30:00,小[改行]
~~~~~~~~~~~~~~~~~~~~~~
のフォーマットになっております。
