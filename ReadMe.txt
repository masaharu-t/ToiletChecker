
ToiletChecker Version 1.00
トイレ時刻チェックアプリ

.NET Framework Visual Studio 2015で開発

アプリを起動して[ 大 ], [ 小 ], [ 大小 ]のボタンをクリックしますとトイレ時刻を記録します。

[ データ追加 ]をクリックしますと手動でトイレ時刻を追加できます。

トイレ時刻一覧の修正したい項目をダブルクリックしますと記録したデータを修正できます。

トイレ時刻一覧のトイレ時刻を削除したい場合は、該当する行を選択して[DEL]キーを押します。


トイレチェックしたデータは、実行ファイル[ToiletChecker.exe]と同じフォルダにあります。
ファイル名は、[ToiletChecker.txt]です。
CSV形式のファイルです。
2016/04/01 06:30:00,小[改行]
~~~~~~~~~~~~~~~~~~~~~~
のフォーマットになっております。
